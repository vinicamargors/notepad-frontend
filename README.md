# notepad-frontend
Frontend da minha aplicação web, um site de anotações integrado ao um banco de dados não-relacional.

## Tecnologias utilizadas:
- React.js
- Html
- Css
- Axios
- Node.js

## Necessário instalar o Node.js e React.js em sua máquina
